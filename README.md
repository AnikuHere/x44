X44 is a custom-built, hardware-integrated language designed for rapid system automation and security protocols.

## Features
- **_** : Kernel Logging
- **>** : Voice Synthesis (TTS)
- **$** : GUI Popups
- **@** : Physical Key Detection
- **&** : Web Downloading
- **?** : Self-Destruct Mode

 X44 is built by Pointware Studios & AmeBits: AME Division, however is based on one of our devs (hello, im aniku/denius!) projects called SC5.

## How to Run
1. Ensure you have **MinGW (g++)** and **Python** installed.
2. Run the build script:

   `python build.py examples/omega.x44`
